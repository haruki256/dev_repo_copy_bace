# feature-TEMPLATE

## 使い方
機能開発時に、このフォルダをコピーして `feature-[機能名]/` にリネームして使う。

## テンプレートファイル
以下のファイルを `doc/AI/flow/template/` からコピーしてこのフォルダに配置する：

| ファイル | 用途 | 作成ステップ |
|---------|------|------------|
| REQUIREMENTS.md | 要件定義 | Step 1 |
| DESIGN.md | 設計書 | Step 2 |
| TESTLIST.md | テストリスト | Step 3 |
| IMPLEMENTATION_PLAN.md | 実装計画 | Step 4 |

テンプレート元: `doc/AI/flow/template/`
開発フロー詳細: `doc/AI/flow/00_flow.md`
