# システム構成ドキュメント

このフォルダには、開発初心者や AI が漏れなく開発に入れるようにするための、
システム全体の構成や前提知識をまとめたドキュメントを配置しています。

## ドキュメント一覧

| ファイル | 内容 |
|---------|------|
| [SYSTEM_OVERVIEW.md](SYSTEM_OVERVIEW.md) | システム全体概要（何を・なぜ・どうやって作っているか） |
| [ARCHITECTURE.md](ARCHITECTURE.md) | アーキテクチャ構成図（モジュール構成・全体設計） |
| [DATA_FLOW.md](DATA_FLOW.md) | データの流れ |
| [TECH_STACK.md](TECH_STACK.md) | 技術スタック詳細と前提知識 |
| [relations/](relations/) | システム関連図・外部システム連携・複数リポジトリ構成方針 |

## 対象読者

- このプロジェクトに初めて参加する開発者
- コードベースの全体像を理解したい人
- AI による開発支援時のコンテキストとして
